# Session 4 Guide — Deploy, Operate & Scale

## Objective

Run Session 4 as a two-part narrative:

1. **What’s new**
   - File-based skills
   - Secrets Vault
   - Job scheduling
2. **Operational flow**
   - Deploy -> Observe -> Automate -> Secure -> Extend (skills) -> Operate at scale

Primary deployment reference: <https://aspire.dev/deployment/>

## Recommended agenda (60 min)

1. Intro and goals (5 min)
2. What’s new in OpenClaw .NET (15 min)
3. Deploy with Aspire options (7 min)
4. Observe + Automate + Secure (12 min)
5. Extend skills + Operate at scale (6 min)
6. Live demo (10 min)
7. Q&A and wrap (5 min)

## Demo checklist

- A sample file-based skill ready for edit/load
- Vault-backed secret resolution working in runtime
- One recurring job and one one-shot job to show
- Deployment option matrix slide (Aspire docs-aligned)
- Observability artifacts (health, traces, metrics)

## Key messages to reinforce

- File-based skills turn agent behavior into reviewable assets
- Vault-backed secrets reduce operational risk and simplify rotation
- Scheduling moves agent workflows from ad-hoc to reliable automation
- Aspire gives a consistent application model while deployment target can vary
- Operations (observe/security/scale) must be designed early, not bolted on
